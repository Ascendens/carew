I'm very happy to release the **second major version** of Carew.

This version come with a lot of new features:

* Built-in pagination.
* Built-in code highlighting.
* Built-in table of content generator.
* More tests.
* More flexible.
* More documentation.
* A better debugging.
* A better design.
